Contact info

Factor Software is in fact a single person, i.e., me, Federico Filipponi. The name Factor Software doesn't refer to any existing company: I just use it as a nickname since 1986.

For any request, suggestion, bug report, etc. you can e-mail me at:

fedefil@factor-software.com

Don't forget to visit my web site at the following URL:

http://www.factor-software.com/

Current Factor Software's products are:

  • BOOM: an arcade game similar to Bomberman.
  • Germs: an action/puzzle game.
  • MacPipes: a classic arcade puzzle in which you have to build a pipeline of required length connecting pipe parts within a time limit.
  • MyEyes: draws on the menubar a pair of eyes constantly following the cursor movement (for Classic Mac OS only).
  • Reanimator: a utility for automatically restarting all processes killed by another process (for Classic Mac OS only).

Please check my web site for up-to-date information and download links.
If you have trouble in retrieving my programs feel free to e-mail me for help.
Support is given via e-mail only.

Newsletter

I'm maintaining a newsletter to notify people of new releases and updates.
To subscribe to the newsletter just send me an e-mail message with Subject: Subscribe me to Factor Software newsletter. Subsequently, you should also inform me of any changes in your e-mail address.
As an alternative, use the simple subscription form you find in the home page of my web site. To remove yourself from the newsletter, use the form in the support page of my web site.
