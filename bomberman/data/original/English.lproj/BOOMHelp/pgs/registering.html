Registering BOOM

When you buy BOOM, you receive from Kagi a purchase confirmation e-mail message which contains your registration information, i.e, your user name and registration code. This code is unique for this product and matches the user name you specify when ordering. Usually this message is sent shortly after purchase to the e-mail address you specified when ordering.
Please save or print your registration information for future reference.

If you buy BOOM using the built-in order form (see "Purchasing BOOM"), the application receives the registration information directly from the order processing system. The registration code is installed automatically and the game is unlocked instantly! No action is required on your part to register the game.

If you buy BOOM online, you need to complete the registration process and unlock your copy of the program manually.
To do so, you have to enter your registration information in the program's registration dialog.
To open the registration dialog, launch the program and click on the "Register..." button in the Shareware Reminder dialog. You can also select the corresponding item from the Application menu.

Both name and code must be typed EXACTLY as reported in the purchase confirmation message.
If your code is rejected, try again. Use copy & paste to avoid errors. If the problem persists, contact me for help (see "Contact info").
