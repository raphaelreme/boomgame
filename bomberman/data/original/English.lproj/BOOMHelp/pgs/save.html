How to save/load games

While playing BOOM you have the opportunity to save a game in progress at any time and restart it later exactly from where you left, with all your lives, points and bonuses intact.

This feature is available to REGISTERED USERS only.

To save a game, just pause (Command-P) and select “Save Current Game…” from the File menu (Command-S).
When in pause, you can also load a previously saved game by selecting “Open Saved Game…” from the File menu (be careful though, because in doing so you will abort the game you were playing).
A saved game can also be loaded from the title screen, either selecting the Open command from the File menu or clicking on the words "OPEN GAME" directly on the title screen.

Saved games are stored automatically in your hard disk in the following folder (the tilde character in the path represents your home folder):

~/Library/Application Support/Factor Software/BOOM

I recommend that you do not manually move, rename or delete the files in this folder. Saved games can be eliminated by using the delete button (-) in the "open game" dialog.
The files corresponding to deleted games are actually moved to the Trash so that you can recover them in case you change your mind.

To recover a deleted saved game file, move it out of the Trash and double click on it directly in the Finder to open it in BOOM. Once the game has started, you can save it again.
