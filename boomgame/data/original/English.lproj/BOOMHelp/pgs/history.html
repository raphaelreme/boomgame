Version history

2.0.5 (September, 2011)

  • Fixed an issue that caused the player to become invincible after losing a life just an instant before advancing to the next level.

2.0.4 (September, 2011)

  • Fixed not working Options menu.
  • Fixed an issue that caused music to not properly stop/resume after disabling/enabling it.
  • A few code enhancements.

2.0.3 (April, 2011)

  • Fixed a bug that caused the game to occasionally crash under some uncommon conditions.
  • Fixed a problem with cursor not auto-hiding on Leopard.
  • A few minor code refinements.

2.0.2 (March, 2011)

  • Various bug fixes and enhancements.

2.0.1 (February, 2011)

  • Fixed a bug that prevented the game to start on Mac OS X 10.5 Leopard.
  • Fixed a problem that occured when a player was losing a life just after completing a level.
  • Fixed font problem on Leopard.

2.0 (February, 2011)

  • The game has been completely rewritten from scratch using Apple's modern development tools and frameworks (Xcode, Cocoa, Objective-C).
  • Runs natively on Intel Macs (Universal Binary).
  • Requires Mac OS X 10.5 Leopard or later.
  • The game engine is now built using MGGame, an OpenGL/OpenAL based 2D animation library developed by Sebastian Wegner.
  • A brand new loading screen has been added, as well as a few new effects and transitions.
  • The game saving feature has been completely redesigned for better usability.
  • Built-in basic USB HID gamepad support (no external plugins/drivers required).
  • Help has been re-engineered and updated.
  • Added Italian localization.
  • Due to the new application architecture, old preferences and saved game files created with a previous version of the game are not compatible with this release.
  • For the same reason, registered users will have to re-enter their registration information to unlock the game.

Release notes for versions prior to 2.0 are available on Factor Software's Web site at the following URL:
