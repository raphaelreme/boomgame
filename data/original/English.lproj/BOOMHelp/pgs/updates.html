Updates

From time to time, Factor Software releases updates to its products. These updates may include both new features and bug fixes.
BOOM gives you the ability to check for updates directly from the game application.

By selecting the "Check for Updates..." command from the Application menu, you can know at any time if a new release of the game is available from Factor Software.
Since the application needs to contact a remote server to retrieve the relevant information, please make sure that your Mac is connected to the Internet before using this functionality.

If you prefer, you can instruct the application to automatically check for updates immediately upon launch.
To activate this feature, open the preferences dialog*, select the "Network" tab and switch on the checkbox labeled "Check for Updates at Startup" (this option is not active in the default configuration).

In case a new version of BOOM is available, you are presented with a dialog displaying detailed information on the update, including version number and release notes.
If you decide to get the new version immediately, click on the "Download..." button and you will be automatically transferred to the download area of Factor Software's web site with your default browser.

*To open the preferences dialog, click on the word "PREFERENCES" on the title screen or select the "Preferences..." item from the Application menu.

Note: during the communication with the remote server, the application does not send any information about you or your system to Factor Software.
